# 学习目录功能总结

## 1. 功能概述

学习目录（LearningPath）是《数据结构》智慧学习平台的核心导航模块，以**左侧树形目录 + 右侧详情内容**的左右分栏布局，为学习者提供清晰的知识结构浏览与精准跳转能力。

## 5. 路由结构

```
/knowledge                         ← LearningPath/index.vue（布局容器）
  /welcome                         ← KnowledgeDetail（默认占位）
  /:module                         ← KnowledgeDetail（按模块浏览）
  /:module/:id                     ← KnowledgeDetail（按知识点ID浏览）
```

点击左侧树节点后，路由跳转 `{ name: 'KnowledgeDetail', params: { module, id } }`，右侧 `<router-view>` 渲染 KnowledgeDetail 页面。

## 7. 技术要点

- **树组件**：Element Plus `el-tree`，`node-key="id"`，`accordion` 手风琴模式
- **搜索过滤**：`filter-node-method` 按 name（中文）和 code（英文）双重匹配
- **状态渲染**：自定义 `#default` 插槽，左侧圆点 + 右侧难度标签 + 前置依赖图标
- **路径引导**：`el-tooltip` hover 显示前置依赖知识点名列表
- **分栏布局**：flex 左右分栏，左侧 320px 固定，右侧 flex:1 自适应
- **全屏适配**：`height: calc(100vh - 56px)` 确保白色背景始终触底

## 9. 接入真实数据步骤

1. **数据层**：填充 MySQL knowledge_point 表 → 导出 Neo4j CSV → 导入节点和关系
2. **后端**：实现 `GET /api/knowledge/tree` → 前端删除 `mockTreeData`，调用 API
3. **后端**：实现 `GET /api/learning/progress` → 前端删除 mockProgress，调用 API
4. **后端**：实现 `GET /api/knowledge/{id}/prerequisites` → 前端可按需调用或保留静态数据
5. **详情**：知识详情负责人填充 KnowledgeDetail/index.vue
